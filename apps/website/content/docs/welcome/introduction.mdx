# Introduction

**Life.js** is a Typescript framework to create interactive experiences with AI agents.

Think of ChatGPT-style chatbots, embedded assistants like Cursor, or Figma-like collaborative editors â€” anywhere users and AI interact using voice, text, or touch.

It works discretely alongside your preferred stack (e.g., Next.js).

<br/>

---

<br/>

In Life.js everything starts with defining an agent.

```ts
// agents/example/server.ts
import { defineAgent } from "life/define";
export default defineAgent("example");
```

<br/>
After running `life dev`, you can interact with your agent from your frontend.

## Features

- End-to-end typesafety
Agents can:
- Perceive the frontend
- Seamlessly combine voice, text, and touch
- Interact with the frontend
- Collaborate in real-time with the user

## Philosophy

The framework has been thought to be as minimal and extensible as possible.

It takes its root from projects like Better Auth, Shadcn, or Bun.
